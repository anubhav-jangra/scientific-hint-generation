Hint Generation Project website!

Garbage Info - HintGen is a prototype for a tool that generates hints for programming problems. It uses GPT-3 to generate hints based on the problem description and the user's code. The prototype is a simple web interface that allows you to enter a problem description and your code, and it will generate hints based on that input.
