# Designing and Evaluating Hint Generation Systems for Science Education

This repository contains the official implementation of out paper:

Designing and Evaluating Hint Generation Systems for Science Education
Anubhav Jangra, Smaranda Muresan
arXiv preprint arXiv:2510.21087 (2025)

## Overview
This project investigates the design and evaluation of automatic hint generation systems for science education. It includes:
- A user interface used for conducting a controlled human-subject study (`UI/`)
- An automatic evaluation framework implementing pedagogically grounded metrics (`autoeval/`)
- The datasets used for both automatic and human evaluation (`data/`)

The codebase is organized to support reproducibility, extension, and further experimentation with hint generation systems.
